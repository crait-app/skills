功能概述

ML Model Debugger 是专为机器学习工程师设计的调试工具，可以帮助你快速定位和解决模型训练和推理过程中的各种问题。

  • 训练诊断: 自动检测梯度消失/爆炸、过拟合等问题
  • 性能分析: 识别模型推理的性能瓶颈
  • 数据验证: 检查数据质量和分布
  • 可视化: 提供丰富的可视化工具
  • 框架支持: 支持 TensorFlow、PyTorch、JAX 等主流框架

适用场景

  • 模型训练不收敛
  • 推理速度过慢
  • 模型性能异常
  • 数据预处理问题

高级特性

实时监控

## 功能概述

- **训练诊断**: 自动检测梯度消失/爆炸、过拟合等问题
- **性能分析**: 识别模型推理的性能瓶颈
- **数据验证**: 检查数据质量和分布
- **可视化**: 提供丰富的可视化工具
- **框架支持**: 支持 TensorFlow、PyTorch、JAX 等主流框架
